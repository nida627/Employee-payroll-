# Employee Payroll System

## 📌 Project Overview

The **Employee Payroll System** is a console-based Python application developed using **Object-Oriented Programming (OOP)** concepts. The system manages employee records and calculates salaries based on different employee roles such as **Manager**, **Full-Time Employee**, and **Intern**.

This project demonstrates the practical implementation of **Abstraction, Inheritance, and Polymorphism** in Python through a menu-driven payroll management system.

---

## 🎯 Objectives

- Learn Object-Oriented Programming (OOP) concepts.
- Implement Abstraction using Abstract Base Class (ABC).
- Understand Inheritance and Polymorphism.
- Calculate employee salaries based on employee type.
- Develop a menu-driven console application.

---

## ✨ Features

- Add Employee
- Remove Employee
- Display All Employees

- Menu-Driven Console Application
- Salary Calculation Based on Employee Role

---

## 📚 OOP Concepts Used

### Abstraction
The `Employee` class is created as an Abstract Base Class using the `abc` module.

### Inheritance
The following classes inherit from the `Employee` class:
- Manager
- FullTimeEmployee
- Intern

### Polymorphism
Each employee class implements its own version of the `calculate_salary()` method.
